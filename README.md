# Kour.io-
kour.io增强版挂：含隐身（invisible）、必杀（‌Instant Kill‌）、伪造信息（Message）等效果

1.隐身（invisible）

随时开关：按Q键开启，Z键关闭

控制台指令：
bash
kourInstance.invisiable=false/true

2.一击必杀（Instant Kill‌）

控制台运行：kourInstance.config.InstantKill = false/true

3.发送信息（Kour.Message）

控制台运行：kourMessager.send("你要发送的信息")

您可在文本中使用<color=red></color>标签更改文本颜色

消息伪造(慎用！)：
kourMessager.send("<color=red>invisibility detected for <color=yellow>XXX")

kourMessager.send("<color=red>XXX <color=green> was<color=purple> punished<color=red> for his sin")
